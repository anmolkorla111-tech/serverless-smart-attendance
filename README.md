## ğŸ— High-Level Architecture
1. User uploads an image
2. Image is stored in an Amazon S3 bucket
3. AWS Lambda function is triggered
4. Amazon Rekognition matches the face
5. Attendance is stored in DynamoDB
6. API Gateway returns the response
