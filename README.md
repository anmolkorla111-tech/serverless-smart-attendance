# Serverless Smart Attendance System

The Serverless Smart Attendance System is a cloud-based attendance management solution
that uses face recognition to automatically mark attendance.

This project is built on a fully serverless architecture, which means:
- No server management required
- Low operational cost
- High scalability and reliability

This project is designed for internships, jobs, and cloud/DevOps portfolios.

---

## Features
- Face recognition–based attendance system
- Fully serverless architecture (no EC2)
- Secure image storage
- Automatic attendance recording
- Scalable and cost-effective
- Resume and portfolio–ready project

---

## Tech Stack
- AWS Lambda (Python)
- Amazon API Gateway
- Amazon Rekognition
- Amazon DynamoDB
- Amazon S3
- IAM (Security & Access Management)
- Git & GitHub

- Offices and organizations
- Training centers
